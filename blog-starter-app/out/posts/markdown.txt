11:T152e,<h2>什么是Markdown？</h2>
<p>Markdown 是一种轻量级的标记语言，设计用来格式化纯文本。它的语法简单易学，便于快速编写格式化文本，特别适用于博客、技术文档、代码注释等场景。最重要的是，Markdown 文本在转换为 HTML 后，可以保持清晰简洁的结构和易读性。</p>
<p>Markdown 在很多平台中被广泛支持，如 GitHub、Gitee、博客平台等。接下来，我们将详细介绍 Markdown 的基本语法。</p>
<h2>1. 标题</h2>
<p>在 Markdown 中，标题通过在文本前加上 <code>#</code> 来定义。<code>#</code> 的个数决定了标题的级别（1 到 6 级标题）。</p>
<h4>例子：</h4>
<pre><code class="language-markdown"># 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
</code></pre>
<h4>渲染效果：</h4>
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
<h2>2. 段落和换行</h2>
<p>Markdown 中的段落通过空行进行分隔。要换行，只需要在行尾加上两个空格后按 <code>Enter</code> 键。</p>
<h4>例子：</h4>
<pre><code class="language-markdown">这是第一段。

这是第二段，换行时需要在行尾添加两个空格。  
这是第二段的后半部分，接在第一行之后。
</code></pre>
<h4>渲染效果：</h4>
<p>这是第一段。</p>
<p>这是第二段，换行时需要在行尾添加两个空格。
这是第二段的后半部分，接在第一行之后。</p>
<h2>3. 强调文本</h2>
<p>可以使用 <code>*</code> 或 <code>_</code> 来强调文本，使其变为 <strong>斜体</strong> 或 <strong>加粗</strong>。</p>
<ul>
<li><strong>斜体</strong>：用单个 <code>*</code> 或 <code>_</code> 包裹文本</li>
<li><strong>加粗</strong>：用两个 <code>**</code> 或 <code>__</code> 包裹文本</li>
</ul>
<h4>例子：</h4>
<pre><code class="language-markdown">*斜体文本* 或 _斜体文本_
**加粗文本** 或 __加粗文本__
</code></pre>
<h4>渲染效果：</h4>
<p><em>斜体文本</em> 或 <em>斜体文本</em>
<strong>加粗文本</strong> 或 <strong>加粗文本</strong></p>
<h2>4. 列表</h2>
<p>Markdown 支持有序列表和无序列表。</p>
<h3>无序列表</h3>
<p>使用 <code>-</code>、<code>*</code> 或 <code>+</code> 来创建无序列表。</p>
<h4>例子：</h4>
<pre><code class="language-markdown">- 项目一
- 项目二
- 项目三
</code></pre>
<h4>渲染效果：</h4>
<ul>
<li>项目一</li>
<li>项目二</li>
<li>项目三</li>
</ul>
<h3>有序列表</h3>
<p>使用数字和点（例如 <code>1.</code>）来创建有序列表。</p>
<h4>例子：</h4>
<pre><code class="language-markdown">1. 第一步
2. 第二步
3. 第三步
</code></pre>
<h4>渲染效果：</h4>
<ol>
<li>第一步</li>
<li>第二步</li>
<li>第三步</li>
</ol>
<h2>5. 链接和图片</h2>
<h3>链接</h3>
<p>创建链接的语法为 <code>[链接文本](链接地址)</code>。</p>
<h4>例子：</h4>
<pre><code class="language-markdown">[Google](https://www.google.com)
</code></pre>
<h4>渲染效果：</h4>
<p><a href="https://www.google.com">Google</a></p>
<h3>图片</h3>
<p>插入图片的语法与链接类似，但前面加上一个 <code>!</code> 符号。</p>
<h4>例子：</h4>
<pre><code class="language-markdown">![图片描述](图片地址)
</code></pre>
<h4>渲染效果：</h4>
<p><img src="https://example.com/image.jpg" alt="图片描述"></p>
<h2>6. 引用</h2>
<p>Markdown 中的引用使用 <code>></code> 来表示。</p>
<h4>例子：</h4>
<pre><code class="language-markdown">> 这是一个引用块。
</code></pre>
<h4>渲染效果：</h4>
<blockquote>
<p>这是一个引用块。</p>
</blockquote>
<h2>7. 代码</h2>
<h3>行内代码</h3>
<p>行内代码使用单个反引号（<code>`</code>）包裹。</p>
<h4>例子：</h4>
<pre><code class="language-markdown">这是 `行内代码` 示例。
</code></pre>
<h4>渲染效果：</h4>
<p>这是 <code>行内代码</code> 示例。</p>
<h3>代码块</h3>
<p>代码块使用三个反引号（```）或者缩进四个空格来标识。你还可以指定代码语言以进行高亮显示。</p>
<h4>例子：</h4>
<pre><code class="language-markdown">```javascript
console.log("Hello, world!");
</code></pre>
<pre><code>
#### 渲染效果：
```javascript
console.log("Hello, world!");
</code></pre>
<h2>8. 水平线</h2>
<p>水平线可以通过三个或更多的 <code>-</code>、<code>*</code> 或 <code>_</code> 来创建。</p>
<h4>例子：</h4>
<pre><code class="language-markdown">---
</code></pre>
<h4>渲染效果：</h4>
<hr>
<h2>9. 表格</h2>
<p>Markdown 支持表格，表格通过 <code>|</code> 来分隔列，<code>-</code> 来分隔表头和内容。</p>
<h4>例子：</h4>
<pre><code class="language-markdown">| 姓名 | 年龄 | 职业 |
| ---- | ---- | ---- |
| Wayne | 28   | 开发工程师 |
| Tom   | 25   | 设计师 |
</code></pre>
<h4>渲染效果：</h4>
<p>| 姓名    | 年龄 | 职业 |
| ----- | -- | ----- |
| Wayne | 28 | 开发工程师 |
| Tom   | 25 | 设计师   |</p>
<h2>总结</h2>
<p>Markdown 是一种轻量级的标记语言，它可以让我们用简单的语法格式化文本，帮助我们编写清晰、结构化的文档。掌握 Markdown 的基本语法后，我们可以轻松创建博客、技术文档、注释等内容。</p>
